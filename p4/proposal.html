DWA Project 4 Proposal

A Recipe builder

by Paul Newell

This site would allow a user to view a set of stored recipes and "execute" them by pressing a button. Here execution means displaying a list of steps and allowing the user to click them done as a recipe is being completed so that they can see what is left (see http://p3.pgnewell.com). The home page would have a list of recipes made public by their builders ranked by some formula using most liked, recent and least viewed and the ability to log in or sign up. Logging in would lead the user to the ability to view or build and save recipes. The user should also be able to affix photographs to the recipes.

The main page would be a divided into 3 sections with a header a sidebar and a display. The header would be mostly decorative. The sidebar would have sign in/up and perhaps something else. The display would have a set of recent entries.
