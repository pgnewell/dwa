DWA Project 3 Proposal

A Recipe builder

by Paul Newell

This project would build a recipe from a pallete of available steps. Steps would include things like bake, stir, cool, etc. but also mis-en-place, which would have associated elements, namely ingredients. For each step dependencies on other steps could be recorded. Recipes would be recorded in XML as would the information related to the steps. You could then execute the recipe, which would display the available steps (the ones with no or completed dependencies) and check them as done.
